1. Install dependencies

GitTor depends on build-essential, libtorrent-rasterbar-dev, libglib2.0-dev, libgit2-dev, libcurl4-openssl-dev, and libjson-glib-dev. 

{{ debianDepsCommand }}

2. Build from source

No prebuilt binaries are published yet, so clone and build:

{{ buildFromSourceCommand }}

The installer may prompt you to add GitTor to your PATH. Once that's done, verify with gittor --help. If you hit issues, the devcontainer config is a good reference for what a known-good environment looks like.

Windows install instructions are being reworked. In the meantime, see the CLI repository for current build steps.
